== Description ==

Hello 420 is a lightweight, minimalist theme optimized for Elementor.

The theme aims to stay lean and stable, providing a clean foundation while keeping customization inside Elementor.

== Requirements ==

* WordPress 6.0+
* PHP 8.2+
* Elementor plugin (recommended)

== Installation ==

1. Upload the theme folder to `/wp-content/themes/hello420/`.
2. Activate the theme in **Appearance → Themes**.
3. Install and activate Elementor.

== Frequently Asked Questions ==

= Does the theme work without Elementor? =

Yes. The theme is activatable and will render standard WordPress templates. Elementor is recommended to unlock full site-building workflows.

= Where are the theme settings? =

Go to **Hello 420 → Settings**.
